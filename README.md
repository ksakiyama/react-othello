# React Example : react-othello
Othello game by React.js. This app is my very first React app.

<img src="app_image.png" width="500px" />

## Setup
```
git clone https://github.com/ksakiyama/react-othello.git
cd react-othello
npm install
npm start
```

## Dependencies
### npm
```
brew install node
```
